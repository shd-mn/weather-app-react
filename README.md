<h2 id='installation'>Installation</h2>

```bash
** open terminal
git clone https://github.com/shd-mn/weather-app-react.git
cd weather-app-react

** then
** install packages
pnpm install
```

<h2 id='starting'>Starting Project</h2>

```bash
pnpm start

** open web page in browser
http://localhost:3000
```
